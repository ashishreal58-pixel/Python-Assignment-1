# Daily Calorie Tracker
This is a small Python project that helps you track how many calories you eat in a day. You can enter your meals, their calories, and a daily calorie limit. The program will then show your total and average calories and tell you if you stayed within your limit or not.

# Features 


1.Add as many meals as you want
2.Calculates total and average calories
3.Warns if you exceed your daily limit
4.Shows a simple daily summary
